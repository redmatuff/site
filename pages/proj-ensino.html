Recursos Computacionais com Python (2023)
Monitor: Gabriel Domingues
Orientador:
Ano: 2023

Ao longo do ano, foi desenvolvido um projeto empregando Python e novas tecnologias com o intuito de explorar, de maneira lúdica, as propriedades das funções trigonométricas. A criação de vídeos visou facilitar e tornar mais atrativo o aprendizado desses conceitos matemáticos para os alunos. A abordagem adotada incluiu programação em Python, utilização de gráficos dinâmicos e recursos visuais para ilustrar as propriedades das funções trigonométricas.
