Recursos Computacionais com Python

Monitores: Gabriel Domingues
Data: 2023

Durante o ano, foi desenvolvido um projeto inovador ao longo do ano, utilizando Python e novas tecnologias para explorar de forma lúdica as propriedades das funções trigonométricas. A criação de vídeos teve como objetivo tornar o aprendizado desses conceitos matemáticos mais acessível e atraente para os alunos. A abordagem envolveu programação em Python, gráficos dinâmicos e recursos visuais para ilustrar as propriedades das funções trigonométricas.
